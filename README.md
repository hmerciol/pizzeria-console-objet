# Pizzeria-Console
Il s'agit du résultat d'une suite de tps sur java. Le contexte général est la gestion du menu d'une pizzeria

## Concepts objets
La première implémentation consistait en une simple architecture objet. Le but était de maîtriser les concepts objets de base de java.

## Manipulation fichiers
La deuxième utilise un fichier local pour stocker le menu. Il s'agissait cette fois de se familiariser avec la manipulation de fichiers locaux avec java.

## JDBC
La troisième utilise la library JDBC. Après un tp sur MySQL, ce tp montrait comment manipuler une base de données MySQL avec java. On se connecte ici sur un serveur de CleverCoud.

## JPA
La dernière implémentation utilise JPA et Hibernate. Suivant le tp Test-Jpa, il est appliqué ici sur le projet pizzeria. On se connecte à la même base que précédemment.
